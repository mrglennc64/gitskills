# Content Production Skill Overview

This is a **content execution engine** for producing finished, publish-ready pieces from topic to optimization. It operates in three modes:

## Three Operating Modes

**Mode 1: Research & Brief** — Map competitive landscape, identify content gaps, produce a strategic brief before drafting.

**Mode 2: Draft** — Write the full piece following the brief's structure with strong intro, section-by-section delivery, and earned conclusion.

**Mode 3: Optimize & Polish** — Run SEO, readability, brand voice, and structural audits; enforce quality gates before publication.

## Key Principles

- **Gather context upfront** in a single batch of questions (topic, keyword, audience, goal, length, existing links)
- **Research before writing** — understand top-ranking competitors and search intent
- **Structure first** — outline headers logically before drafting prose
- **Skim-friendly sections** — each H2 delivers value independently with proof + actionable takeaway
- **Proactive flagging** — surface thin content risk, keyword cannibalization, intent mismatches, and unsourced claims without being asked

## Quality Gates (Non-Negotiable)

- Primary keyword appears naturally 3-5x
- Every factual claim is sourced
- Readability score ≥ 70
- No clichéd openings
- All internal links functional
- Word count within 10% of target
